SmartCo - Employee & Project Management System
SmartCo is a full-stack web application for managing employees, projects, tasks, permissions, notifications, attendance, payroll, and reports.

This project was built as a final practical software engineering project using Angular, PHP, and MySQL.

Main Features
User login and session management
Role-based permissions
Employee management
Project management
Task creation and assignment
Task status tracking
Internal task chat
Notifications system
Attendance tracking
Payroll calculation
Payslip PDF generation
Dashboard and reports
